HTML Basics

Please select a tag:

<div>

The <div> tag is a generic container used to group elements together. It has no semantic meaning, but is useful for styling and layout. It is often combined with CSS to organize a webpage.

<p>

The <p> tag is used for paragraphs of text. Browsers automatically add some margin around it for readability. It helps structure written content in a clear and accessible way.

<h1>

The <h1> tag represents the highest level heading. It is very important for SEO and page structure. Proper use of heading tags improves accessibility and search engine rankings.

<a>

The <a> tag creates hyperlinks. Use the href attribute to specify the destination URL. Links allow navigation between pages, and they can also open files, trigger actions, or link to different sections within the same document.

<img>

The <img> tag embeds images. It requires the src attribute to define the image source and the alt attribute for accessibility. The alt text is important for screen readers and helps when images fail to load.

<ul>
  • Uses <li> for each list item.
  • Displays items with bullet points by default.
  • Great for lists where order doesn't matter.
<ol>
  1. Uses <li> to define list items.
  2. Items are automatically numbered.
  3. Great for step-by-step guides and rankings.
<table>
Tag Reason to Use Basic Syntax
<table> Defines the table structure <table>...</table>
<tr> Defines a table row <tr>...</tr>
<th> Defines a header cell <th>Header</th>
<td> Defines a standard cell <td>Data</td>
<form>

The <form> tag is used to collect user input. It contains input fields like textboxes, checkboxes, and buttons. Forms allow users to submit data, such as login credentials, feedback, or search queries, to a server.
